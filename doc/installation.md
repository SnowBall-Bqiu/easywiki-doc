# 开发指南

## 开发步骤

1. 克隆仓库
```bash
git clone https://github.com/SnowBall-Bqiu/easywiki.git
```

2. 安装tailwindcss

3. 启动开发服务器
```bash
npm run dev
```

## 配置选项

编辑`sidebar.json`文件可自定义导航菜单:
```json
[
  {"title": "首页", "link": "home.md"}
]
```
